Como fazer funcionar manualmente de forma local:
Tenha o "Node.js" instalado.
No terminal, digite os comandos:

1. "cd "C:\Users\Seu\Caminho\marketplace-condominio\marketplace-condominio-api""
2. "npm init -y"
3. "npm install"
4. "npm install express cors body-parser"
5. "node server.js"
6. Pronto!
